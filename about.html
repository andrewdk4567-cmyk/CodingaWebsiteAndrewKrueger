Welcome!

This website was created by Andrew Krueger as a student project for ARTD 245: Web Design at North Central College in Naperville, IL, in 2025. All the text comes from the following articles located on the Dribbble website. All the images were created by me using Photoshop. The website has been created using HTML and CSS in Dreamweaver.

About Me

My name is Andrew Krueger. I’m a design major at North Central College with a passion for graphic design and marketing. I enjoy creating digital art, posters, and animation projects that help ideas come to life through visual storytelling.

Website Purpose

This website shows how different design principles work together to make a great looking and functional website. It was built using HTML and CSS in Dreamweaver to demonstrate layout, color, and typography in a practical way.
